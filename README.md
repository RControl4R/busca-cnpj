# Busca de CNPJ

## Case:
Uma página web para trazer informações de um determinado CNPJ que foi inputado. Utilizando como mecanismo de busca o https://brasilapi.com.br/ , visa contornar o uso através de páginas como Sintegra, que normalmente pedem autenticação Captcha para cada requisição, o que gera atrasos para o colaborador.


## Tecnologias utilizadas:
- JavaScript (https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
- NodeJS (https://nodejs.org/en)
- Express (https://expressjs.com/pt-br/)

## Hospedagem:
- Render (https://render.com/)
